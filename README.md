# 🤖 AI Interview System

An AI-powered interview preparation and evaluation system built using Python and Streamlit.  
It helps candidates practice HR interviews, get instant feedback, and analyze performance like a real ATS system.

---

## 🚀 Features

- 🎤 Voice-based answer input (Speech Recognition)
- 🧠 AI-powered interview evaluation
- 📄 Resume (PDF) upload and text extraction
- 📊 Automatic performance scoring
- 💬 HR-style feedback (communication, confidence, improvement tips)
- 📁 Save interview results to Excel
- 🖥️ Simple and interactive Streamlit UI
- 📈 Ready for Power BI dashboard integration

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Google Generative AI (Gemini API)
- SpeechRecognition
- PyPDF2
- Pandas
- OpenPyXL
